# Static search index (local dev)

The app loads a search index from `VITE_SEARCH_BASE_URL` (default `/data`).

For local development, copy the generated index files here:

```
data/teams.json   ->  frontend/public/data/teams.json
data/events.json  ->  frontend/public/data/events.json
```

These files are generated by `data/generate_teams_search.py` and
`data/generate_events_search.py` at the repo root.

In production, host these on a CDN/static bucket and point
`VITE_SEARCH_BASE_URL` at it instead of bundling them.

The `.json` files here are gitignored on purpose (they are large and rebuilt by
the pipeline).
